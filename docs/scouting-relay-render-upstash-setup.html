PowerScout relay setup

New Render + Upstash Relay Server

This checklist is for the new scouting-only relay server using the team account. It is not the old DirectChat or The Button setup. The goal is a clean relay endpoint for PowerScout, the Firebase scouting website, and scout devices.

What Render runs

scouting-relay

A small Node service inside the scouting repo. It exposes health checks and scout pager APIs.

What Upstash stores

Redis message state

Event-scoped head-scout pings, scout inbox messages, and short-lived relay state.

What stays local

Secrets

Tokens go in Render environment variables, never in Firebase-hosted client code or screenshots.

After Deploy: Verify It Works

Render should give you this service URL if the service name is powerscout-relay:

https://powerscout-relay.onrender.com

Health check

curl -sS https://powerscout-relay.onrender.com/health

Expected result:

{
  "ok": true,
  "service": "powerscout-relay",
  "runtime": "render-upstash",
  "storage": "upstash"
}

If storage says memory, the Upstash environment variables are missing or wrong.

Admin stats check

curl -sS \
  -H "Authorization: Bearer PASTE_THE_ACTUAL_ADMIN_TOKEN_HERE" \
  "https://powerscout-relay.onrender.com/api/pager/stats?eventKey=2026CASNV"

Do not paste the text PASTE_THE_ACTUAL_ADMIN_TOKEN_HERE. Replace it with the real Render environment variable value for POWERSCOUT_RELAY_ADMIN_TOKEN.

Send one test scout message

curl -sS -X POST \
  -H "Authorization: Bearer PASTE_THE_ACTUAL_ADMIN_TOKEN_HERE" \
  -H "Content-Type: application/json" \
  -d '{
    "eventKey": "2026CASNV",
    "recipient": { "kind": "scout", "scoutNumber": 7, "scoutName": "Test Scout" },
    "title": "Relay test",
    "body": "If this appears, PowerScout relay messages are working.",
    "ttlMs": 600000
  }' \
  https://powerscout-relay.onrender.com/api/pager/messages

Read the test message as Scout #7

curl -sS \
  -H "Authorization: Bearer PASTE_THE_ACTUAL_SCOUT_TOKEN_HERE" \
  "https://powerscout-relay.onrender.com/api/pager/messages?eventKey=2026CASNV&scoutNumber=7"

Do not paste the text PASTE_THE_ACTUAL_SCOUT_TOKEN_HERE. Replace it with the real Render environment variable value for POWERSCOUT_RELAY_SCOUT_TOKEN.
