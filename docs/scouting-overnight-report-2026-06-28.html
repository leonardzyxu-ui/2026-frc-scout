How To Use It During Competition

  1. Start the day: open Admin V4, choose the competition phase, and follow the Now-screen action.
  2. Practice matches: use practice forecasts, save Forecast Snapshots, and export the workbook after each block.
  3. Qualifications: use Forecast Ledger rows to compare what the model knew before matches against actual outcomes.
  4. Alliance selection: mark Our Pick, Taken, Declined, Unavailable, or Clear immediately after each public pick.
  5. Visitors or judges: press Shift+D, then show Model Demo Proof, Model Trust, Visualize, and Admin V2 Prediction vs Actual.
